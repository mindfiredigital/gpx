## Shell Integration
- Supports Zsh, Bash & PowerShell
### Commands
### Setup shell hooks

### Zsh
```
gpx init --shell zsh
```

### Bash
```
gpx init --shell bash
```

### PowerShell
```
gpx init --shell powershell
```

**What it adds:**
- Prompt badge: `~/projects [<active_profile_name>] $`
- cd-hook: auto-detect on directory change (if enabled)
- Prompt updates on `gpx use`
